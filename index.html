Website

Pengembangan aplikasi berbasis web, dari awal pengerjaan aplikasi berupa konsultasi desain, sampai tahap akhir berupa pengujian hosting / server, domain aplikasi sehingga dapat digunakan oleh end user.

Mobile Apps

Desain dan Pengembangan aplikasi berbasis mobile app : Android. Mengintegrasikan mobile app dengan API .

Fitur

Desain aplikasi.

Merancang desain UI/User Interface aplikasi yang menarik dengan UX/User eXperience yang mudah digunakan.

  • Reponsive design. Desain aplikasi yang akan disesuaikan dengan ukuran layar/monitor Anda.
  • Modern design framework. Berbagai pilihan untuk modern desain UI Anda : bootstrap, semantic ui, Material Design

Agile & Continuous Deployment

Continuous Flow pengerjaan dan release / deployment aplikasi.

  • Agile Scrum, Kanban. Daily update untuk progress pengerjaan aplikasi menggunakan framework agile dan metodologi SCRUM, Kanban.
  • Git Repository. Source code aplikasi yang disimpan dengan sistem versioning, memudahkan untuk mengecek history perubahan code.
  • Test. Testing aplikasi menggunakan unit test (mengecek logic aplikasi), penetration test (mengecek kemampuan aplikasi menghandle banyak requests).
  • Continuous Deployment. Release perubahan code ke server hosting secara otomatis terintegrasi dengan git repository.

Produk E-Government

SMART TNDE

Pengelolaan Naskah Dinas Elektronik (Inbox & Outbox).

  • Web & Mobile Based. Aplikasi dapat diakses melalui Web Based maupun Mobile App Yang Responsif.
  • Form Based. Aplikasi dapat membuat naskah dinas secara langsung.

SMART PLAN & BUDGETING

Perencanaan Program Kerja dan Anggaran

  • Smart Plan. Aplikasi menangani proses perencanaan program kerja dan anggaran mulai dari awal sampai dengan disahkan secara definitif termasuk revisi pada tahun berjalan.
  • Smart Tracking. Aplikasi dapat mentrack perubahan rencana program kerja dan anggaran yang direncanakan maupun berjalan.
  • Smart Input. Aplikasi dapat secara otomatis mengisi pos pos anggaran sesuai dengan SBU / SBK atau Historis sebelumnya .
  • Smart Preventive. Aplikasi dapat mencegah terjadinya potensi markup ataupun potensi tidak optimalnya anggaran.
  • Smart Manage. Aplikasi dilengkapi dengan kemampuan membuat Project Management.
